# EE217 - Embedded Systems

Course materials and lab work for EE217.

## Contents

### HW3 - Signal Processing
Python implementations for homework assignments covering:
- **PRBS (Pseudo-Random Binary Sequence)** generation and analysis
- **CDMA (Code Division Multiple Access)** demonstrations
- **Autocorrelation and cross-correlation** analysis
- **Touch sensor** signal processing and noise estimation
- **Sampling, aliasing, and DTFT** concepts

### Labs

#### Lab1
Python environment setup for embedded systems development.

#### Lab2
High-Precision AD/DA Board experiments with Raspberry Pi.

## Requirements

- Python 3.x
- NumPy
- Matplotlib

## Setup

```bash
# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install numpy matplotlib
```
