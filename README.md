# EE217 - Embedded Systems

Course materials and lab work for EE217.

## Contents

### Lab2 - High-Precision AD/DA Board
Demo code for the High-Precision AD/DA Board with support for:
- **ADS1256** - 24-bit ADC
- **DAC8532** - 16-bit DAC

Platforms supported:
- Raspberry Pi (bcm2835, wiringPi, Python)
- Jetson Nano

## Setup

Refer to the readme files in each platform directory for installation and usage instructions.
